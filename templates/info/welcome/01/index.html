01 - Minecraft Server Information

Ok, so let's clear some basic information first.

Our server address is yjjcity.ddns.net (Same as the website!)

If you don't know how to add a server, here are the instructions.
1. In the Main Menu in Minecraft, click on Multiplayer.
2. Click on Add Server at the bottom right.
3. Fill in the blanks as shown in the picture below. Then click Done and it will appear on the Server List.

Our server is open 24/7! But just to be sure, check our Discord to see if it is online.

Below is a video to show you how to check if the server is online. Here are the steps:

1. Go to #general in the YJJ City Discord
2. Click on the text at the top of the screen.
3. See the LAST PING message.
4. If the last ping is more than 10 minutes from the current time, the server is down.
(YJJ City uses Singapore General Time (GMT+8). You can check the time at the Home page of the YJJ City Site.)
